Introduction to localStorage and sessionStorage

localStorage and sessionStorage accomplish the exact same thing with the same API. Both have the following rules:
• They store value/key pairs for a particular domain.
• The values are stored as strings.
The difference is that when data in localStorage doesn't expire. but when data is loaded in sessionStorage it will be expired after closing the browser. Local storage saves our data for future use but session storage data for while we are browsing the website. Like we can use Facebook without login and something different type of online shop save our data.
